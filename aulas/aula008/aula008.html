Principais formatações

Negrito

Tem duas tags

<b>

Essa tag é uma forma não semantica de deixar uma frase em Negrito

<strong>

Já essa tag é a forma semantica de deixar uma frase em Negrito

diferança

As duas tags quando se olha no site não muda muito, mas assim a tag <strong> ela da enfase a frase pois ela aumenta um pouco a palavra e da mais força a palavra.

Italico

Existe duas tags

<i>

A tag <i> é uma forma não semantica, uma tag que está ficando obsoleta de deixar uma frase em Italico

<em>

A tag <em> é a forma semantica de deixar uma frase em Italico

diferança

A olho nu talvez você não perceba diferança, pois não muda nada na hora de olhar, mas sim muda na HTML, pois a tag <i> está ficando obsoleta, e em um futuro próximo ela ira parar de funcionar.

Texto deletado e Texto inserido

Texto deletado

Para criar um texto deletado basta usar a tag <del>

Texto inserido

Para criar um Texto inserido basta usar a tag <ins>

Tambem tem outra forma de sublinhar um texto mas é uma forma não semantica, utilizando a tag <u>

Diferença

Visualmente não é possivel ver a diferança entre eles. Texto 1Texto 2 a diferança é que o texto 2 no caso a tag <u> é uma forma não semantica, e já está ficando obsoleta.

Texto sobescrito e Texto subscrito

Texto sobescrito

o texto sobescrito serve para deixar uma palavra mais alta que outra um exemplo é em numeros, como por exemplo é cinco elevado a vinte, 520 = 520 para fazer isso basta usar a tag <sup>.

Texto subscrito

O texto subscrito serve para deixar a palavra um pouco para baixo, um exemplo é h2o = h2o para fazer isso basta usar a tag <sub>.
