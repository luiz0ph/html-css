Código fonte e pré-formatações

<code>

A tag <code> serve para dar o mesmo espaçamento para todos os caracteres, é muito util para quando vai se ler em algum site.

<pre>

A tag <pre> ele deixa no site do mesmo jeito que está no código, sem nenhuma formatação

Exemplo:
        num = int(input("Escolha um número: "))
        if num == 2
            print("Seu número é dois")
        else
            print("Seu número não é dois")
